# Catch of the Day
A menu ordering platform for various dishes of fish, built with React

### Contains:
- `Authentication`: Facebook, Twitter, Github Social Auth via Firebase
- `Database`: Firebase Realtime Database
- `Build`: Gulp
- `State Rehydration`: LocalStorage
